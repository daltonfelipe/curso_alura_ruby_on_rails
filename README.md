# Steps to use this app

* In terminal change directory for project e.g. `$ cd project_folder/`
* Install project dependecies with `$ bundle install`
* Run app with `$ rails server`

Things you may want to cover:

* Ruby version **2.5.1**
* Rails version **5.2.3**
